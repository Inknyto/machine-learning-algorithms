# Arbres de Décision

Ce répertoire contient une implémentation des arbres de décision pour la classification.

## Fichiers
- `decision_tree_titanic.ipynb` : Utilise un classificateur d'arbre de décision pour prédire la survie des passagers du Titanic.

## Données
- `titanic.csv` : Jeu de données contenant des informations sur les passagers du Titanic (âge, sexe, classe, etc.) et s'ils ont survécu.
