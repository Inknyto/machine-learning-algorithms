# Support Vector Machine (SVM)

Ce répertoire contient des applications des machines à vecteurs de support (SVM).

## Fichiers
- `svm_sales_prediction.py` / `svm_sales_prediction.ipynb` : Prédiction des quantités de vente (`QTE_VENTE`).
- `svm_user_classification.py` / `svm_user_classification.ipynb` : Classification d'utilisateurs basée sur l'âge et le salaire estimé.

## Données
- `VenteCommerciale.csv` : Données de transactions commerciales.
- `User_data .csv` : Données démographiques et comportementales des utilisateurs.
