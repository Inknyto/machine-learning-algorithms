# Gradient Boosting

Ce répertoire contient des algorithmes de Gradient Boosting pour la régression.

## Fichiers
- `gradient_boosting_real_estate.ipynb` : Prédiction des prix de l'immobilier au Maroc.
  - Nettoyage des données (gestion des valeurs manquantes, suppression des zéros).
  - Encodage des variables catégorielles (emplacement).
  - Utilisation du Gradient Boosting pour la prédiction du "prix en MAD".

## Données
- `Location_Data.csv` : Données sur les locations immobilières, incluant l'emplacement, le nombre de chambres, de salons, la surface et le prix.
