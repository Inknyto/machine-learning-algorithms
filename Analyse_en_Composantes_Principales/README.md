# Analyse en Composantes Principales (ACP / PCA)

Ce répertoire contient des implémentations de l'Analyse en Composantes Principales (ACP), une technique de réduction de dimensionnalité non supervisée.

## Fichiers
- `ACP.ipynb` : Implémentation standard de l'ACP sur le jeu de données Iris.
  - Utilise `StandardScaler` pour la normalisation.
  - Visualisations en 2D et 3D des données projetées.
- `pca_iris_tutorial.ipynb` : Tutoriel détaillé basé sur le "Python Data Science Handbook" de Jake VanderPlas.
  - Explore l'ACP comme outil de visualisation et de réduction de dimensionnalité.
  - Utilise Seaborn pour des graphiques avancés.

## Données
- `Iris.csv` : Le célèbre jeu de données Iris contenant des mesures de sépales et de pétales pour trois espèces de fleurs.
