# Naive Bayes

Ce répertoire contient des implémentations de l'algorithme Naive Bayes pour la classification.

## Fichiers
- `naive_bayes_titanic.py` / `naive_bayes_titanic.ipynb` : Comparaison de différentes variantes de Naive Bayes (Gaussian, Multinomial, Categorical) pour prédire la survie sur le Titanic.
  - Prétraitement des données : suppression des colonnes non pertinentes et encodage des variables.

## Données
- `titanic.csv` : Jeu de données des passagers du Titanic.
