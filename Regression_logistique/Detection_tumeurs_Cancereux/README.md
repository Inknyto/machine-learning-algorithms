# Détection de Tumeurs Cancéreuses

Ce sous-répertoire se concentre sur l'application de la régression logistique pour le diagnostic médical.

## Fichiers
- `breast_cancer_analysis.py` / `breast_cancer_analysis.ipynb` : Analyse complète des données, calcul des corrélations et visualisation par heatmap.
- `breast_cancer_model.py` : Version épurée du script pour l'entraînement et l'évaluation du modèle.
- `graphe_RL.py` : Scripts dédiés aux visualisations graphiques de la régression logistique.

## Données
- `breast-cancer.csv` : Jeu de données contenant des caractéristiques de tumeurs mammaires et leur diagnostic (Malin ou Bénin).
