# Détection de Tumeurs Cancéreuses

Ce sous-répertoire se concentre sur l'application de la régression logistique pour le diagnostic médical.

## Fichiers
- `breast_cancer_analysis.ipynb` : Analyse complète des données, calcul des corrélations et visualisation par heatmap.
- `breast_cancer_model.ipynb` : Script d'entraînement et d'évaluation du modèle.

## Données
- `breast-cancer.csv` : Jeu de données contenant des caractéristiques de tumeurs mammaires et leur diagnostic (Malin ou Bénin).
