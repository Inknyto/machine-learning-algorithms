# Régression Logistique

Ce répertoire contient des implémentations de régression logistique pour la classification binaire.

## Fichiers
- `logistic_regression_cancer.py` / `logistic_regression_cancer.ipynb` : Modèle de base pour la détection du cancer du sein.
- `Detection_tumeurs_Cancereux/` : Sous-répertoire dédié à l'analyse plus approfondie.
  - `breast_cancer_analysis.py` : Exploration des données, corrélations et visualisations.
  - `breast_cancer_model.py` : Script simplifié pour l'entraînement du modèle.
  - `graphe_RL.py` : Scripts de visualisation spécifiques.

## Données
- `breast-cancer.csv` : Jeu de données sur les tumeurs mammaires (caractéristiques des cellules et diagnostic).
